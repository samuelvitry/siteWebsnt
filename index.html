Les posibilitées du html 5

Le html 5 est la dernière mise a jour du language HTML. Elle a été finalisée le 28 octobre 2014.
Elle apporte beaucoup de nouveautées. Certaines de ces nouveautées permettent même de créer des pages web en 3D.
Un des meilleurs exemple de ce genre de page est le jeu Cube slam.
Pour faciliter la création de ce genre de page l'éditeur de jeu unity a décidé d'incorporer un exporteur HTML5 dans sa dernière mise a jour.
